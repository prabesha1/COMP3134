Q1: What is the definition of server port?
A server port is a numerical identifier used in networking to designate a specific process or service running on a computer. It ensures that data transmitted over a network reaches the correct application on the server.

Q2: What is server port range?
The range of port numbers is from 0 to 65,535. These are categorized into three groups:
- System Ports (Well-Known Ports): 0 to 1023
- User Ports (Registered Ports): 1024 to 49151
- Dynamic/Private Ports: 49152 to 65535

Q3: How many bits does number above represent?
Port numbers are represented by 16 bits (specifically, a 16-bit unsigned integer).
